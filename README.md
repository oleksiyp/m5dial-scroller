
Software to turn [M5 Dial](https://docs.m5stack.com/en/core/M5Dial) to
USB HID device to scroll browser.

Gives ability:
 - scroll with dial
 - select controls with dial (flaky)
 - press on controls
 - refresh browser
 - go back/forward
